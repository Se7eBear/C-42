# C-42: Fundamentos e Lógica em C 🚀

Este repositório é dedicado ao estudo aprofundado da Linguagem C, servindo como base de preparação para a **Piscine da 42 São Paulo**. O foco aqui é o domínio da manipulação de memória e lógica de baixo nível.

## 📌 Sobre o Projeto
O repositório documenta a progressão dos meus estudos, partindo de operações básicas até a gestão direta de memória com ponteiros. A organização segue uma ordem lógica de aprendizagem para facilitar a revisão de conceitos fundamentais da ciência da computação.

---

## 📂 Estrutura de Arquivos

A estrutura foi refatorada para utilizar nomes técnicos que descrevem o conceito explorado em cada módulo:

* **`00_input_arithmetic.c`**: Implementação de fluxos de entrada e saída de dados, validação de input com `scanf` e operações aritméticas.
* **`01_conditional_logic.c`**: Estudo de estruturas de controle e tomada de decisão através de blocos condicionais (`if/else`).
* **`02_memory_address.c`**: Exploração física da memória, utilizando o operador de endereço (`&`) e o formatador `%p` para mapeamento no Linux.
* **`03_pointer_basics.c`**: Introdução aos ponteiros, abordando a atribuição de endereços e a técnica de desreferenciação (`*ptr`) para acesso a dados.

## 📈 Registro de Evolução Diária
Este repositório é atualizado diariamente com novos exercícios, refatorações e lógicas de resolução. O objetivo é manter uma trilha contínua de commits que reflita o progresso real e a consistência nos estudos de baixo nível, preparando o fluxo de trabalho para a intensidade da Piscine.
